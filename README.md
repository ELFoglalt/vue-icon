# vui-icon
vue2 svg icon component

图标来自[手机淘宝图标库](http://iconfont.cn/plus/collections/detail?cid=33)

# Installation
```
$ npm install vui-icon
```
# Usage
``` xml
<vui-icon name="close"></vui-icon>
```
